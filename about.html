About


Truthy or Falsey is a quiz based website that allows for children to supplement their educational needs with a fun environment in which they can test themselves on various topics. An easily usable navigation menu has been set up at the top of the page to allow for adults to intuitively move around the website whilst children can press on links within the landing page in order to explore content. The actual quizzes are set up so that children only need to interface through the press of a button which should give them immediate feedback.


Rules

  1. Click on the prompt in the landing page to access the quiz
  2. The question will appear when the quiz page appears. Simply press one of the green buttons to answer
  3. Squigzy the squirrel will let you know if the answer is correct
  4. Once the answer has been given then press on the Next Question button to bring up the next question
  5. Once 10 questions have been answered a Game Over prompt will appear and give you a tally of correct answers
  6. To replay just refresh the window
